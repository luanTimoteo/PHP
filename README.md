A ideia é ir evoluindo aos poucos e deixar tudo registrado pra acompanhar meu progresso e também usar como portfólio futuramente.

---

## 🚀 O que tem aqui

Nesse projeto eu comecei vendo o básico de PHP, usando junto com HTML.

Algumas coisas que aprendi até agora:

- Criar variáveis usando $
- Exibir informações na tela com echo
- Quebra de linha com <br>
- Uso de constantes com define()
- Arrays (vetores) com vários tipos de dados
- Arrays associativos (com chave e valor)
- Matrizes (arrays dentro de arrays)
- Uso de var_dump() pra ver mais detalhes das variáveis
